# Learner

Learner is a Geometry Dash practice tool for routing start positions, guided section training, and cleaner repeat attempts.

## Features

- Switch start positions with keybinds or on-screen arrows during practice.
- Guided Mode trains sections in phases before expanding into longer runs.
- Smart Startpos syncs nearby portal states for accurate practice attempts.
- Per-level settings keep guided, smart, display, cutoff, and attempt options separate.
- Music sync keeps startpos attempts aligned with the correct song timing.
- Pause menu progress shows the current phase, route, and guided stage count.
